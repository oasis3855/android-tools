## 概要

環境省の大気汚染物質広域監視システム「そらまめ君」がAPIとして公開しているjsonデータを読み込んで画面に表示するアプリ。 

<br />

2023年現在、環境省のシステムのAPIが変更されているため、**このプログラムは利用できない**が、「AsyncTaskクラスを利用した**非同期処理**」や「org.json.JSONObjectクラスを用いた**JSONデータへのアクセス**」の参考資料となるため、残存させている。
